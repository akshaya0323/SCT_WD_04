# 🌌 Galaxy To-Do List

A modern and visually appealing **To-Do List Web App** with a galaxy-inspired UI.
Organize your daily tasks with a clean interface, glowing elements, and a space-themed background.

---

## 🚀 Features

* ➕ Add tasks with date & time
* ✅ Mark tasks as completed
* ❌ Delete tasks
* 🔍 Filter tasks:

  * All
  * Pending
  * Completed
  * Today
  * Overdue
* 📊 Task statistics (Total, Done, Pending)
* 🌌 Galaxy UI with starry background
* 💎 Glassmorphism design
* 📱 Fully responsive layout

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling (Glassmorphism + Galaxy Theme)
* **JavaScript (Vanilla JS)** – Functionality

---

## 🎯 How It Works

* Tasks are stored in a temporary array (`tasks[]`)
* Each task contains:

  * id
  * text
  * completed status
  * date & time
* Filters dynamically update the UI
* Stats update automatically after each action
